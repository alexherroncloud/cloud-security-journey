## File type & permissions breakdown

at first sight, when I seen the output of [ls -l] it threw me off lol. So I wanted to make a note of what the output means.

-drwxr-x--- 21 greenray greenray 4096 jun 6 11:32 greenray

#file type & permissions
- the - (dash) at the start means the item is a reg file.
- d (drwxr-x) : the d means the file is a directory.
# File Ownership
- 21 : number of hardlinks to the file
- greenray greenray : user & group that owns the file.
#file size
- 4096 : the file size in bytes
#date and time
- jun 6 11:32 : last modification time, updated/created.
#Name
- The final column shows the name of file or folder/directory.

#note 
This is just a detailed inventory list.
